## Dockerfile

```Dockerfile
FROM cgr.dev/chainguard/go AS build


COPY main.go /main.go
RUN CGO_ENABLED=0 go build -o /hello /main.go


FROM cgr.dev/chainguard/static
COPY --from=build /hello /usr/local/bin/
CMD ["hello"]
```
