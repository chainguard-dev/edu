## Variants Compared
The **gradle** Chainguard Image currently has one public variant: 

- `latest`

The table has detailed information about each of these variants.

|              | latest            |
|--------------|-------------------|
| Default User | `gradle`          |
| Entrypoint   | `/usr/bin/gradle` |
| CMD          | not specified     |
| Workdir      | `/home/build`     |
| Has apk?     | no                |
| Has a shell? | yes               |
