Configure a CMake project

### Dependencies
- cmake
- ninja


### Reference
| Input        | Description                                                          |
|--------------|----------------------------------------------------------------------|
| `output-dir` | The output directory for the CMake build. Default is set to `output` |
| `opts`       | Compile options for the CMake build.                                 |


### Example
```yaml
uses: cmake/configure

```
