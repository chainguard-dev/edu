Configure project with meson

### Dependencies
- meson


### Reference
| Input        | Description                                                          |
|--------------|----------------------------------------------------------------------|
| `output-dir` | The output directory for the Meson build. Default is set to `output` |
| `opts`       | Compile options for the Meson build.                                 |


### Example
```yaml
uses: meson/configure

```
