Apply patches

### Dependencies
- patch


### Reference
| Input              | Description                                                                    |
|--------------------|--------------------------------------------------------------------------------|
| `strip-components` | The number of path components to strip while extracting. Default is set to `1` |
| `patches`*         | A list of patches to apply, as a whitespace delimited string.                  |


### Example
```yaml
- uses: patch
    with:
      patches: libtool-fix-cross-compile.patch
```
