Run the autoconf configure script

### Dependencies
None

### Reference
| Input | Description                                                          |
|-------|----------------------------------------------------------------------|
| `dir` | The directory containing the configure script. Default is set to `.` |


### Example
```yaml
- uses: autoconf/configure
  with:
    opts: |
      PYTHON=/usr/bin/python3 \
      --with-lzma \
      --with-zlib

```
